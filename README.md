# SDVX-Panel-Conversion
Turn an arcade Sound Voltex panel into a controller.

Obviously a WIP.  Requires a 1x2ft laser cutter, a 3d printer, and an assortment of m3 and m4 nuts and bolts.  And rubber feet.
